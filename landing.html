End-To-End Data Analytics Project

Web scraping football data from understat in Python - Tranforming data with SQL - Creating advanced Vizualisations in Tableau

Introduction

This is an end-to-end data analytics project where I used a variety of tools and techniques to extract, transform, and visualize data. To begin, I utilized web scraping in Python to gather raw data from Understat, a website that provides detailed football statistics. As a huge football fan, I was excited to have the opportunity to work with this data and see different insights in Europes top 5 leagues. Next, I used SQL to transform the data and prepare it for analysis. Finally, I used Tableau to create informative data visualizations. If you're interested in seeing the final product, be sure to check out the data visualization dashboards. You will find it below.
